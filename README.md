# 전국 경비업체 목록 자동 수집

네이버 검색(지역검색) API를 이용해 전국 229개 시·군·구의 경비업체 정보(업체명, 주소, 전화번호, 이메일, 홈페이지)를 자동으로 수집하고, 매주 자동 갱신하는 프로젝트입니다.

## 결과 파일

- 위치: [`data/전국_경비업체_목록.xlsx`](data/전국_경비업체_목록.xlsx)
- 매주 월요일 오전 9시(한국시간)에 자동 갱신됩니다.
- 재수집 시 기존 데이터는 삭제되지 않고 업데이트/누적됩니다.

### 컬럼 설명

| 컬럼 | 설명 |
|---|---|
| 업체명 | 네이버 검색에 등록된 상호명 |
| 주소 | 도로명 주소 (없으면 지번 주소) |
| 전화번호 | 대표 전화번호 |
| 이메일 | 홈페이지가 있는 경우 첫 페이지에서 찾은 공개 이메일 (없으면 공란) |
| 홈페이지 | 네이버 링크가 아닌 외부 홈페이지만 표시 |
| 최초수집일 | 이 업체가 처음 수집된 날짜 |
| 최종수정일 | 정보가 마지막으로 변경된 날짜 |

## 지금 바로 새로 수집하고 싶을 때 (수동 실행)

1. 상단 메뉴 **Actions** 클릭
2. 왼쪽에서 **"경비업체 목록 자동 수집"** 클릭
3. 오른쪽 위 **Run workflow** 버튼 → 드롭다운에서 초록색 **Run workflow** 클릭
4. 새 실행 번호가 생성되고 진행 상황을 실시간으로 확인 가능 (완료까지 약 20~40분)

> ⚠️ **"Re-run jobs" 버튼은 사용하지 마세요.** 이 버튼은 실행이 처음 시작될 당시의 워크플로 설정을 그대로 재사용하기 때문에, 이후 코드/워크플로를 수정해도 반영되지 않습니다. 반드시 위 3번 방법(새 Run workflow)으로 실행하세요.

## 실행 상태 아이콘

| 아이콘 | 의미 |
|---|---|
| 🟠 주황 원 | 진행 중 (기다리면 됨) |
| ✅ 초록 체크 | 성공 |
| ❌ 빨간 X | 실패 — 클릭해서 "collect" job의 오류 단계를 펼쳐 로그 확인 |

## 구성 파일

- `경비업체_수집기.py`: 수집·필터링·엑셀 저장을 수행하는 파이썬 스크립트
- `.github/workflows/collect.yml`: 매주 자동 실행 + 결과 자동 커밋을 담당하는 GitHub Actions 워크플로

## 데이터 수집 방식과 한계

- 경비업 허가는 시·도경찰청(지방경찰청) 소관이라 지방행정 인허가데이터(localdata.go.kr)에는 포함되지 않고, 공공데이터포털에는 지역별 **집계 통계**만 공개되어 있어 개별 업체명이 담긴 공식 공공데이터가 존재하지 않습니다.
- 이 때문에 네이버 지역검색 API로 "시/도 + 구/군 + 키워드(경비업체·시설경비·보안업체)" 조합을 반복 검색해 업체 정보를 수집합니다.
- 공식 등록 전수 조사는 아니며, 실제 영업 중이고 네이버에 노출된 업체 위주로 수집됩니다.
- 카테고리·업체명 기준으로 열쇠업체, 경찰관서, CCTV설치업체, 탐정사무소 등 관련 없는 결과는 자동으로 걸러내지만, 100% 완벽하지는 않을 수 있습니다.

## 필요한 사전 설정 (이미 완료됨)

- 저장소 **Settings → Secrets and variables → Actions**에 아래 두 개가 등록되어 있어야 합니다.
  - `NAVER_CLIENT_ID`
  - `NAVER_CLIENT_SECRET`
- 네이버 개발자센터([developers.naver.com](https://developers.naver.com))에서 애플리케이션을 등록하고 '검색' API를 사용 설정하면 발급받을 수 있습니다.

## 문제 해결

| 증상 | 확인할 것 |
|---|---|
| 실행이 실패(빨간 X) | Actions → 해당 실행 → collect → 빨간 X 단계를 펼쳐 오류 메시지 확인 |
| 401 오류 (API 인증 실패) | Settings → Secrets에서 `NAVER_CLIENT_ID`/`NAVER_CLIENT_SECRET` 값이 정확한지 재확인 (앞뒤 공백 주의) |
| push 거부 오류 (rejected) | "Re-run jobs"가 아니라 새 "Run workflow"로 실행 |

## 지역 확장하는 법

`경비업체_수집기.py`의 `TARGET_REGIONS` 딕셔너리에 시/도와 그 안의 구/군을 추가하면 됩니다.

```python
TARGET_REGIONS = {
    "서울": ["종로구", "중구", ...],
    "부산": ["해운대구", "부산진구", ...],
}
```

현재는 전국 17개 시·도, 229개 시·군·구 전체가 이미 포함되어 있습니다.
